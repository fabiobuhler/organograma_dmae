DMAE ORGANOGRAMA - SISTEMA DE GOVERNANÇA OPERACIONAL
==================================================

Versão: 1.0.2026.04270145
Desenvolvedor: Fábio Bühler

Este sistema consolidado gerencia a estrutura organizacional do DMAE, 
integrando pessoas, ativos de contingência, contratos e BI administrativo.

DIRETRIZES DE PRODUÇÃO:
- A fonte da verdade é o Supabase.
- Logs de auditoria são permanentes.
- A centralização da árvore é garantida via Safe Centering.
- Ativos de contingência exigem contato direto via WhatsApp.

REPOSITÓRIO:
