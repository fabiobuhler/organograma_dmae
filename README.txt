DMAE - Sistema de Organograma e Gestao de Ativos

Sistema avancado para visualizacao da estrutura organizacional do DMAE, integrado com gestao de ativos, pessoal e contratos de contingencia.

FUNCIONALIDADES PRINCIPAIS
- Visualizacao Dinamica de Organograma (Modos Arvore e Lista)
- Gestao de Ativos Centralizada (com suporte a fotos e Plano de Contingencia)
- Gestao de Pessoal e Contratos vinculados
- Governanca e Seguranca (3 Niveis de Acesso: Publico, Editor, Admin)
- Monitoramento Real-time via Supabase Presence
- Estatisticas de BI e Auditoria Imutavel
- Exportacao para Excel (CSV), PDF e Relatorios de Ativos

TECNOLOGIAS
- React.js, Vite, Vanilla CSS
- Supabase (PostgreSQL + Realtime)
- Lucide-React, html2canvas, jspdf

REQUISITOS DO BANCO (SUPABASE)
Tabelas: nodes, assets, persons, contracts, users, audit_logs, asset_types.

DESENVOLVIMENTO
1. npm install
2. npm run dev (Ambiente de Desenvolvimento)
3. npm run build (Gerar Pacote de Producao)
